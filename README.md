## Controls
  BASIC CONTROLS:
 
 * Left and Right Arrow to move the Flippers
 * Down Arrow to prepare the Kicker
 * Let go of Down Arrow to throw the kicker
 * When Dead: Up Arrow to Restart
 
  DEBUG KEYS:
 
 * F1: To Show Hitboxes and also enable mouse interaction
